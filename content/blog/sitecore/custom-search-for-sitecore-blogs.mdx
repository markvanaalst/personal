Whenever I use Google to search for Sitecore information I get a lot of marketing and CMS review results and a few results with detailed developer information.
So I created a [Google custom search engine](http://www.google.com/coop/cse/ 'Google Custom Search') to search through all Sitecore related blogs. Now I can
search through sites I think are the most usefull for me as an developer.

I use it a lot and thought I might be usefull to share this the rest of the community. So I created a little ASP.Net application around it which makes use of
the [Google Custom Search API](http://code.google.com/apis/customsearch/ 'Google Custom Search API'). Now it is possible for everyone to submit an url of which
they think it's usefull. I allready included all blogs which are used in the [Yahoo Pipe](http://pipes.yahoo.com/sitecore 'Sitecore Yahoo Pipe'). I hope that
this search engine will be very usefull for everyone. I am planning on creating an search provider for both Internet Explorer and Firefox in the near future.

You can find the custom search engine at: [http://www.sitecoreblogsearch.com](http://www.sitecoreblogsearch.com 'Sitecore Blog Search')

And off course you can use the comments to let me know what you think of the custom search.
