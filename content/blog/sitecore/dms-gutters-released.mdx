Today I published a new client feature on the marketplace called DMS Gutters. This will install two gutters within the Content Editor that shows which items
have running tests and personalized components. It will help the business user identifying which items contains tests and personalisation.

<Img src="/images/blog/dmsgutters-featured.png" />

You can download the installation package via the Sitecore Marketplace.

You can find it [here](http://marketplace.sitecore.net/en/Modules/DMS_Gutters.aspx "Sitecore Marketplace")
